Restrições honestas

Disponibilidade, backups testados e PCN atualizado continuam dependentes de fonte externa. Não aparecem aqui como número calculado.

Próximo clique natural

Se a dúvida for “como medir”, o caminho é `Indicadores`. Se a dúvida for “quem responde”, o caminho é `RACI`. Se a dúvida for “como executar”, o caminho é `POPs`.
